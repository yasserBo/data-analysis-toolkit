# User Guide

## Introduction

This Data Analysis Toolkit is designed for beginners who want to explore, clean, visualize, and statistically analyze datasets using Python.

## How to Use the Toolkit

1. Open the notebook `Data_Analysis_Toolkit.ipynb`.
2. Upload a CSV dataset or use the sample dataset.
3. Run the notebook cells step by step.
4. Start with data exploration.
5. Check missing values.
6. Clean the dataset.
7. Generate descriptive statistics.
8. Create visualizations.
9. Analyze probability distributions.
10. Run statistical inference tests.
11. Export the cleaned dataset and results.

## Main Features

- Dataset import
- Missing-value analysis
- Data cleaning
- Summary statistics
- Histograms
- Boxplots
- Scatter plots
- PDF and CDF plots
- Normality testing
- Confidence intervals
- Hypothesis testing
- Exporting results

## Sample Dataset

The example dataset used in this toolkit is the Students Performance in Exams dataset.

It contains columns such as:

- gender
- race/ethnicity
- parental level of education
- lunch
- test preparation course
- math score
- reading score
- writing score

## Example Analysis

The toolkit can be used to analyze student exam scores, compare score distributions, check whether scores are normally distributed, and perform hypothesis tests.
